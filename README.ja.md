## Exile Creator Kitとは？

Exile Creator Kit（ECK）は、

**Path of Exile / Path of Exile 2 プレイヤー向け**

に開発された、

無料・オープンソースの動画エクスポートツールです。

FFmpegの知識は必要ありません。

動画を選択して

**X用**

または

**YouTube用**

をクリックするだけで投稿用動画を作成できます。

---

# こんな人におすすめ

- ボス討伐動画を投稿したい
- ビルド紹介動画を作りたい
- Xへすぐ投稿したい
- YouTubeへ高画質で投稿したい
- FFmpegは難しくて使いたくない
- 動画編集よりゲームを楽しみたい

---

# 主な機能

- X向け動画エクスポート（512MB最適化）
- YouTube高画質エクスポート
- ドラッグ＆ドロップ対応
- 自動ビットレート調整
- 動画情報の自動取得
- FFmpeg同梱
- エクスポート履歴
- Windowsデスクトップアプリ
- 完全無料
- オープンソース（MIT）

---

# 使い方

1. 最新版をダウンロード
2. ZIPを展開
3. ExileCreatorKit.exe を起動
4. 動画を選択
5. X または YouTube をクリック

以上です。
